hyPACK-2013 Hands On Session


hyPACK-2013 Hands-on Sessions (HoS) will be conducted on HPC Cluster with coprocessors and accelerators. Programming on ARM Processor Systems, NVIDIA ARAM Processor system with CUDA enabled NVIDIA carma and DSP Multi-Core processor systems for Mode-1, Mode-2, Mode-3, Mode-4, & Mode-5 modules. The heterogeneous programming is used for development of application kernels and numerical linear algebra (NLA) on hybrid computing systems ( HPC Cluster with coprocessors & acclerators ). in Mode-1, Mode-2, Mode-3, Mode-3 & Mode-5 modules of hypack-2013 are given below.

Laboratory Session Acess & Programming Environment

  • Users can access Computing Systems with various HPC GPU Accelerators and Intel Xeon-Phi Coprocessors supporting various programming paradgms on HOST-CPU and GPU Accelerators & coprocessors.

  • On Each Day-1, users can compile and execute codes based on programming paradigms as given below.

      • HOST CPU :
      • MPI
      • OpenMP
      • Pthreads
      GPU :
      • NVIDIA-PGI-OpenACC
      • NVIDIA CUDA / OpenCL
      • OpenCL
      • AMD OpenCL

      Coprocessor :
      • Intel Xeon-Phi Compiler Vectorization
      • Intel Xeon-Phi Offload Pragmas
      • MPI & OpenMP 3.0 /4.0
      • Cilk Plus & OpenCL
