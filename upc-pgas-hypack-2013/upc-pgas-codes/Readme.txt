pie_cal1.upc: 
		UPC Program, here each Thread calculates the Value of pie using Monte-Carlo Simulation.

pie_cal2.upc:
		MPI-UPC Program each calculates the Value of pie using 	Monte-Carlo Simulation,Result is Average of individual values calculated by each Thread .

jacobi.upc:	
		UPC-Program ,gives the Solution of System of Linear equations satisfying Diagonal Dominance , using Jacobis method.The Diagonal Dominence matrix is 			Randomly Generated by the Program.

matrix_mul.upc:
		MPI-UPC Program, performs matrix multiplication of two given Matrices.Matrices are Randomly Generated.

poisson.upc:  
		MPI-UPC Program for Poisson 2-d Solver.

vect_mul:
		MPI_UPC Program for Vector-Vector Multiplication.Both the Vectors are Randomly Generated.

stream.upc:
		MPI-UPC Program , for measuring Performance of various Memory Read and Write Operations (STREAM BENCHMARK). 	




Compiling:
	1)UPC Program

		Compile UPC Program using :  upcc filename.upc	

		Example: upcc poisson.upc

	2)MPI-UPC Program
		
		upcc -netwrok=mpi -uses-mpi -link-with=mpicc filename.upc <PATH TO FILE mpi.h > -lmpich
		
		Example:upcc -network=mpi -uses-mpi -link-with=mpicc poisson.upc -I/usr/local/mpich2-1.0.7/include/ -lmpich 

Executing:
	
	upcrun -n <Number of threads> ./a.out
	
	example: upcrun -n 8 ./a.out
